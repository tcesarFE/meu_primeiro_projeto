Tutorial de HTML5

O que é o HTML5?

HTML (Hyper Text Markup Language - Linguagem de Marcação de Hipertexto), como o próprio nome já diz, é uma linguagem de marcação de texto. Ela é utilizada principalmente na estruturação de sites da web. A mesma é responsável por apresentar conteúdos te texto, imanges, vídeos, cabeçalhos, tabelas e outros elementos estruturais.

O número 5 na sigla HTML5, refere-se à versão mais recente da linguagem de marcação.

Tags, Elementos e Atributos

Na prática, a estruturação de um site com HTML, utiliza ferramentas chamadas: tags, elementos e atributos. A função de cada um deles é a seguinte:

  • Tags - Contém instruções para exibir algum elemento na página. Elas começam com um < e terminam com um >. (Por exemplo: <b>Marca o elemento em negrito</b>)
  • Elementos - São os conteúdos inserigo nas tags. (<b>Elementos</b>)
  • Atributos - São informações adicionais inseridas dentro das tags e adicionam funcionalidades extras às mesmas. (<html lang="pt-br">)
