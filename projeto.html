O que é o HTML5

HTML (Hyper Text Markup Language - Linguagem de Marcação de Hipertexto), como o próprio nome já diz, é uma linguagem de marcação de texto. Ela é utilizada principalmente na estruturação de sites da web. A mesma é responsável por apresentar conteúdos te texto, imanges, vídeos, cabeçalhos, tabelas e outros elementos estruturais.

O número 5 na sigla HTML5, refere-se à versão mais recente da linguagem de marcação.
